This template is retired.

The yco-spec builder (spec_builder.py) owns all rendering — there is no hand-authored HTML template anymore. The old section-oriented HTML that used to live here was the retired v1 layout and is no longer the spec format.

To build a spec, copy cfg-template.py into your feature's Project/<feature>/build_spec.py, fill in the cfg dict, and run it. See SKILL.md for the full schema and phases.
